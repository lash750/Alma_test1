Описание алгоритма работы функции F(text), принимающей строку и возвращающей ее в обратном порядке.

Пременные :

строка s_currant;
строка s_new;
массив символов arr_char;
целое count_char;

Алгоритм:

1. передаем функции F строку s_currant;
2. вычисляем длину строки s_currant и присваиваем coun_char;
3. заполняем массив arr_char символами с первого по coun_char из s_currant;
4. в цикле создаем новую строку s_new помещая в нее символы из массива arr_char в обратном порядке 
for(count_char; count_char>=0; count_char--)
{
 s_new .= arr-char[count_char];
}
5. возвращаем s_new;
